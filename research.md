## Рекомендованный дизайн-стек

### Шрифты
**Основная пара:** `Raleway 700` (заголовки) + `Inter 400` (тело)

- Тёплая: `Montserrat 700` + `Lato 400`
- Академическая: `Playfair Display 700` + `DM Sans 400`

**Типографическая иерархия:**

### Психологические и кризисные (Mind UK, NAMI, Samaritans, Crisis Text Line)

|---|---|---|

### Крупные фонды (Gates, Ford, Wellcome, UNICEF, MSF)
- Много воздуха, мало цветов
- Serif в заголовках = авторитет (Ford Foundation, Wellcome)
- Числа и статистика в крупных цифрах — обязательная полоса под hero
- Footer: номер регистрации, годовые отчёты, партнёры

## 7 обязательных паттернов доверия

1. **Телефон в шапке** — на каждой странице, крупно, всегда
2. **H1 обращён к человеку** — «Вы не одиноки. Мы здесь.», не «Мы предоставляем услуги»
3. **Числа как якорь** — `X лет | X человек | X консультаций` с countup-анимацией
4. **Реальные лица с именами** — не стоковые. Фото команды с именем и специализацией
5. **Конфиденциальность** — отдельный блок рядом с формой записи (критично для психологии)
6. **Прозрачность** — номер регистрации АНО, устав, ссылки на документы (в России снимает подозрения)
7. **Воздух в дизайне** — щедрые отступы. Для аудитории в трудной ситуации whitespace = ощущение безопасности

## Структура лендинга

1. **Шапка** — логотип + телефон крупно + кнопка «Записаться»
2. **Hero** — Centered minimal. Большой заголовок, 2 строки подзаголовка, CTA «Получить бесплатную консультацию»
3. **Услуги** — 4 карточки с цветной дифференциацией по направлениям
4. **Как мы работаем** — 3 шага (снижает тревогу перед первым обращением)
5. **Цифры доверия** — X лет / X человек / X консультаций
6. **Команда** — фото + имена + специализация
7. **Истории клиентов** — анонимные нарративы (имя изменено + 1 абзац)
8. **Прозрачность** — устав АНО, лицензии, партнёры
9. **CTA-секция** — яркая, оранжевая кнопка
10. **Контакты + форма** — максимум 3 поля + блок конфиденциальности

## Экспертная оценка

> Составлена 2026-04-18. Три независимых взгляда на концепцию лендинга АНО «Прометей Консалтинг».

---

### ЭКСПЕРТ 1 — UX-дизайнер (10 лет практики)

#### Типичные ошибки новичков в структуре

1. **Несколько равновесных CTA** — когда кнопок «Записаться», «Подробнее», «Позвонить» одинаковый вес, пользователь не понимает, куда идти. Одно главное действие на экран.
2. **Доверие — в самом низу** — новички прячут регистрационные данные и отзывы в footer. Для психологической НКО доверие должно появиться на 2-м экране, не на 9-м.
3. **Форма без контекста безопасности** — форма висит отдельно, без объяснения «что будет после». Пользователь в кризисе не заполнит её без чёткого «мы перезвоним в течение X часов, конфиденциально».
4. **Мобиль как последний шаг** — верстают для десктопа, потом «адаптируют». У целевой аудитории (люди в трудной ситуации) смартфон — основное устройство. Начинать надо с mobile-first.
5. **Sticky-шапка без телефона** — телефон должен быть виден в любой точке прокрутки. В исследовании это есть в рекомендациях, важно не потерять при реализации.
6. **Анимации как украшение** — fade-in и countup хороши, но если они тормозят загрузку или мерцают на старых телефонах — хуже, чем без них.

#### Обязательно на лендинге

| Блок | Почему критично |
|------|-----------------|
| Телефон в sticky-шапке | Часть аудитории вообще не заполняет формы — только звонит |
| H1 про проблему посетителя, не про организацию | «Вы не одни» работает, «Мы предоставляем услуги» — нет |
| Форма максимум 3 поля + кнопка «Получить консультацию» | Каждое лишнее поле = -15–20% конверсии |
| Блок конфиденциальности рядом с формой | Психологическая НКО — без этого форму не заполнят |
| Реальные фото команды с именами | Анонимная организация = недоверие |
| Шаги «Как это работает» | Снимает тревогу перед первым обращением |
| Footer с регистрационными данными | Номер АНО, устав, отчёты — снимает подозрения «это секта?» |

#### Что раздражает и заставляет уйти

- Поп-ап на первых 3 секундах («Подпишитесь!», «Не уходите!»)
- Автовоспроизведение видео или звука
- Медленная загрузка — >3 сек = потеря части аудитории на мобиле
- Слайдер с автопрокруткой (уже в «Избегать» — правильно)
- Несколько конкурирующих CTA одного веса
- Стоковые улыбающиеся фото «счастливых людей» — вызывают отчуждение у человека в кризисе
- Текст блоками по 6–8 строк без абзацев — глаз не цепляется, уходят

### ЭКСПЕРТ 2 — Маркетолог по конверсии

#### Оффер, который зацепит клиента

Целевая аудитория — люди в уязвимом состоянии. Им нужно снять три страха одновременно: **«это дорого»**, **«узнают»**, **«не поможет»**.

**Текущий оффер в исследовании:** «Получить бесплатную консультацию» — правильное направление.

**Усиленная версия:**
> «Первая консультация — бесплатно. Анонимно. Без обязательств.»

Три слова снимают три барьера — именно так работает Crisis Text Line («Free. 24/7. Confidential.»). Это уже отмечено в исследовании, важно вынести в H1 или подзаголовок hero, а не прятать в блок услуг.

**Что ещё усиливает оффер:**
- Конкретика времени: «Ответим в течение 2 часов»
- Снижение обязательства: «Просто поговорим — без диагнозов и давления»
- Социальное доказательство рядом с CTA: «Уже 1 200 человек обратились к нам»

#### Триггеры доверия (приоритет)

1. **Цифры с контекстом** — не просто «10 лет», а «10 лет помогаем людям в СПб выйти из кризиса»
2. **Анонимная история клиента** — 1 абзац реального нарратива до CTA-секции меняет конверсию сильнее всего остального
3. **Лица + специализация команды** — «Анна, психолог-кризисолог, 8 лет практики» > просто фото
4. **Блок прозрачности** — номер АНО, ссылка на устав, годовой отчёт. Для России это критично — снимает «что-то мутное»
5. **Гарантия конфиденциальности** — отдельный визуальный элемент (иконка замка, текст «Мы не передаём данные»), не просто строчка в форме

#### Путь посетителя

```
Попал на страницу (реклама / рекомендация)
    ↓
Hero: «Вы не одни» — почувствовал, что это про него [3–5 сек]
    ↓
Блок услуг: нашёл своё направление [10–20 сек]
    ↓
«Как это работает» — 3 шага: понял, что не страшно [30 сек]
    ↓
Цифры доверия + история клиента: поверил [1–2 мин]
    ↓
Команда: увидел живых людей — доверие выросло [1 мин]
    ↓
CTA-секция: оффер + форма 3 поля + конфиденциальность [заполняет]
    ↓
Подтверждение: «Мы свяжемся с вами в течение 2 часов» [спокойствие]
```

**Критическая точка потери:** между «блок услуг» и «CTA». Если нет истории клиента и блока «Как мы работаем» — большинство уходят именно здесь.

#### Что отсутствует в текущем исследовании

- Нет работы с возражением **«я не псих / я справлюсь сам»** — нужен микротекст в hero или рядом с CTA: «Обратиться за помощью — это сила, не слабость»
- Нет сценария для **повторного посещения** — ретаргетинг, pixel, или хотя бы «подпишитесь на полезные материалы»
- **Бесплатность** нигде не выведена визуально крупно — это главный оффер, он должен быть в 3–4 местах на странице

### ЭКСПЕРТ 3 — Потенциальный посетитель

*Ситуация: человек ищет психологическую помощь в трудный момент, открыл сайт с телефона.*

#### Первые 5 секунд — что он видит и думает

| Вопрос в голове | Ответ должен быть на экране |
|---|---|
| «Это для меня?» | H1 про его ситуацию: «Трудно. Мы рядом.» |
| «Это реальные люди?» | Фото команды или живое фото в hero |
| «Это платно?» | «Первая консультация — бесплатно» крупно и видно |
| «Меня не осудят?» | Слово «конфиденциально» в hero или шапке |
| «Как с ними связаться?» | Телефон крупно в шапке |

Если хотя бы на 3 из 5 вопросов нет ответа — вкладка закрыта.

#### Что заставит остаться

- H1, в котором он узнаёт свою ситуацию — не «мы предоставляем психологические услуги», а «Когда трудно — важно не оставаться одному»
- Живые фото команды: реальные лица снимают страх «я попаду к незнакомцу»
- Чёткий ответ на вопрос «что будет, если я нажму кнопку» — шаг 1, шаг 2, шаг 3
- Анонимная история человека, который был в похожей ситуации и обратился
- Отсутствие навязчивости: никаких поп-апов, никакого «Осталось 3 места!»
- Ощущение пространства — воздух в дизайне = ощущение безопасности (уже есть в исследовании, это верно)

#### Что заставит закрыть вкладку

- Корпоративный язык: «комплексные решения», «профессиональный подход»
- Стоковые фото улыбающихся людей — выглядит фальшиво, когда тебе плохо
- Непонятно: это платно или бесплатно? — сомнение убивает
- Форма с 6+ полями: имя, фамилия, телефон, email, тема, удобное время — уйдёт
- Ничего о конфиденциальности до формы — страшно оставлять данные
- Медленная загрузка на мобиле

### Общий список рекомендаций по итогам трёх экспертов

#### Критично (делать обязательно)

1. H1 = про чувство/ситуацию посетителя, не про организацию
2. Слова «бесплатно» + «конфиденциально» — крупно, в hero, видно без прокрутки
3. Телефон в sticky-шапке — всегда виден
4. Форма — максимум 3 поля: имя, телефон, кнопка
5. Блок конфиденциальности — визуальный элемент прямо у формы
6. Реальные фото команды с именами и специализацией
7. Блок «Как мы работаем» — 3 простых шага до первого обращения
8. Анонимная история клиента — 1 абзац до CTA-секции
9. Номер регистрации АНО в footer + ссылка на устав

#### Важно (сильно влияет на конверсию)

10. Микротекст «Обратиться за помощью — это сила» — снимает стыд
11. Цифры с контекстом: не «10 лет», а «10 лет помогаем людям в СПб»
12. «Ответим в течение 2 часов» — конкретика после кнопки CTA
13. Mobile-first верстка — начинать с телефона, не адаптировать потом
14. Загрузка <3 сек — обязательно, аудитория на мобиле
15. Нет поп-апов, автоплея, агрессивных триггеров срочности

#### Желательно (влияет на качество)

16. Оглавление в начале документа research.md (для навигации)
17. Hover и focus состояния для форм — accessibility
18. Alt-тексты для всех изображений — WCAG, SEO
19. Медиа-брейкпоинты: 480px, 768px, 1024px (сейчас только размеры шрифтов)
20. Компоненты: мобильное меню (hamburger), модальное окно, поля форм

## Тренды 2024-2025

### Актуально
- **Humanist Sans:** Inter, Plus Jakarta Sans, DM Sans, Nunito
- **Serif + Sans пара:** Playfair Display + DM Sans; Lora + Source Sans 3
- **Шалфейный зелёный:** `#4A7C59`, `#7FB069` (не агрессивный «эко»-зелёный)
- **Тёплая земля:** терракота `#C97D4E`, охра — для организаций работающих с людьми
- **Документальная фотография** реальных людей > любые иллюстрации
- **Scroll fade-in** анимации (AOS.js), countup для чисел, hover translateY(-2px)

### Избегать
| Что | Почему |
|-----|--------|
| Стоковые улыбающиеся офисные фото | Мёртвый тренд, вызывает недоверие |
| Generic flat-иллюстрации (Undraw-стиль) | Перенасыщен рынок, выглядит дёшево |
| Агрессивный красный `#FF0000` как основной | Тревога, опасность — не для психологических НКО |
| Стерильный белый + синий (больничный) | Стресс для аудитории психологического центра |
| Слайдеры с автопрокруткой | Устаревший паттерн |
| Яркий оранжевый как основной цвет | Слишком «продающий», не вызывает доверия у уязвимых |
| Loading screens | Убивают конверсию |

## ЧАСТЬ 1: Психологические и кризисные НКО

### 1. Mental Health America — mhanational.org

**Палитра:**
- Основной: зелёный `#4CAF50` / `#2E7D32`
- Вторичный: тёмно-синий `#1A237E`
- Акцент: `#8BC34A`
- Фон: `#FFFFFF`, `#F5F5F5`
- Текст: `#212121`, `#616161`

**Шрифты:** Montserrat Bold (заголовки) + Open Sans Regular (тело). H1: 42–56px, body: 16–18px, line-height: 1.6–1.8

**Hero-структура:** Текст слева 60% / фото справа 40%. Под hero — полоса с числами.

**Элементы доверия:** числа ("1 в 5 американцев"), год основания (1909), логотипы корпораций-доноров, цитаты с фото.

**CTA:** прямоугольник, радиус 4–6px. Цвет: `#4CAF50`, белый текст. Текст: глагол + конкретика ("Find Support", "Get Screened").

### 2. NAMI — nami.org

**Палитра:**
- Основной: сине-зелёный `#00A19A`
- Вторичный: `#1C3F6E`

**Элементы доверия:** телефон горячей линии в шапке на КАЖДОЙ странице. GuideStar/Charity Navigator оценки.

**CTA:** pill-shape, радиус 20–24px. Основной: `#00A19A`. Экстренный: жёлтый `#FFB300`.

### 3. Crisis Text Line — crisistextline.org

**Палитра:**
- Основной: лазурный `#0076B6`
- Акцент экстренный: `#E8002D`
- Тёплый акцент: `#FF6B35`

**CTA:** pill-shape, радиус 50px. Экстренный: `#E8002D`.

## ЧАСТЬ 2: Крупные международные фонды

### Gates Foundation
- Чистый, корпоративный минимализм. Белый + тёмный шрифт + оранжевый акцент.
- Шрифт: Gotham / собственная гарнитура. Много пространства, мало цвета.

### Wellcome Trust
- Нестандартный, смелый дизайн. Чёрный/тёмно-синий + яркий акцент (красный, оранжевый).
- Авторитетный, академический тон.

### Ford Foundation
- Тёмно-синий `#003366` + белый + минимальный акцент.
- Serif в заголовках — Garamond-style. Авторитет через типографику.

### Open Society Foundations
- Синий `#1A4FCF` + белый. Чистый, институциональный.

### Charity: Water (лучший дизайн среди всех)
- Жёлтый `#FFC423` + почти чёрный `#1A1A1A` + кремовый `#F5F0EB`.
- Принцип: "снизь барьер входа, покажи результат". Сильные истории + точные числа.
- Кастомный шрифт Knockout (заголовки) + Ideal Sans (тело).

## ЧАСТЬ 6: Тренды 2024-2025

### Цветовые тренды
1. **Доверительный нейтрал + один акцент** — тёплые бежи/офф-вайты + один сильный цвет
2. **Глубокий синий/индиго** — `#1B3A5C`, `#2563EB` (смещение от ярко-синего к «умным» оттенкам)
3. **Шалфейный зелёный** — `#4A7C59`, `#7FB069` (не агрессивный экологический)
4. **Тёплая земля** — терракота, охра, `#C97D4E`, `#D4956A` — для организаций с людьми
5. **Монохромный с акцентом** — тёмный фон + яркий акцент (для tech-oriented аудитории)

### Типографические тренды
- **Humanist Sans** — Inter, Plus Jakarta Sans, DM Sans, Nunito
- **Serif + Sans пара** — Playfair Display + DM Sans; Lora + Source Sans 3
- Крупные заголовки (80–120px) с умеренным font-weight 400–500
- line-height: 1.6–1.8 для тела (аудитория в стрессе = читаемость)

### Иллюстрации vs фото
- **Уходит:** стоковые фото улыбающихся офисных людей, flat-иллюстрации Undraw-стиля
- **Актуально:** документальные живые фото, абстрактные органические формы, line art, размытие/маскировка для защиты приватности

### Layout hero-секции (рекомендация для «Прометея»)
**Centered minimal** — всё по центру, большой заголовок, 1–2 кнопки + карточки услуг сразу под ним.

### Анимации
- Scroll fade-in (AOS.js или CSS) — duration 500–700ms
- Countup для статистики
- Hover на кнопках (translateY -1–2px)
- НИКАКИХ: слайдеры с автопрокруткой, loading screens, параллакс агрессивный

### Что ИЗБЕГАТЬ в НКО-дизайне
| Элемент | Почему |
|---------|--------|
| Агрессивный красный `#FF0000` | Тревога, опасность — только для emergency-организаций |
| Кислотные/неоновые цвета | Дешёвые продажи, не для серьёзных тем |
| Золотой как основной | Ассоциация с роскошью — противоречит НКО |
| Ярко-оранжевый как основной | Слишком «рекламный», не вызывает доверия у уязвимых |
| Стерильный белый + синий | «Больничный» — стресс для психологических НКО |
| Generic flat-иллюстрации | Перенасыщен рынок, выглядит дёшево |

## ЧАСТЬ 7: 7 обязательных паттернов доверия

1. **Телефон в шапке** — на каждой странице, крупно
2. **H1 обращён к человеку** — не «Мы предоставляем услуги», а «Вы не одиноки. Мы здесь.»
3. **Числа как якорь** — `X лет | X человек | X консультаций`
4. **Реальные лица с именами** — не стоковые, с именем и 1 фразой
5. **Блок «в кризисе прямо сейчас?»** — отдельно, не смешивать с остальным
6. **Прозрачность** — номер регистрации АНО, устав, ссылки на документы (особенно важно в России)
7. **Белое пространство** — воздух в дизайне = ощущение безопасности

## ЧАСТЬ 8: Структура лендинга «Прометей Консалтинг»

1. **Hero** — Centered minimal. «Поддержка, которая меняет жизни» + подзаголовок + CTA «Получить бесплатную консультацию»
2. **Услуги** — 4 карточки с цветной дифференциацией
3. **Как мы работаем** — 3 шага (снижение тревоги)
4. **Цифры доверия** — countup анимация
5. **Команда** — фото + имена + специализация
6. **Истории клиентов** — анонимные нарративы
7. **Документы / прозрачность** — устав, лицензии АНО
8. **Партнёры** — логотипы
9. **Контакты + форма записи** — максимум 3 поля + блок конфиденциальности

### Особые требования для психологической НКО
- Блок конфиденциальности рядом с формой
- Мягкие переходы — без резких границ
- Никакого давления в CTA («Начните с бесплатной консультации», не «Срочно запишитесь»)
- Мобильная версия — приоритет (многие обратятся в момент кризиса с телефона)
